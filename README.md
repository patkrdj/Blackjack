# 블랙잭 게임

블랙잭 카드 게임을 구현한 프로젝트입니다. 딜러와 플레이어가 21에 가장 가까운 숫자를 만들어 승부를 겨루는 카지노 게임을 시뮬레이션합니다.

## 게임 규칙

### 기본 규칙
- 딜러와 플레이어는 카드의 합이 21에 가까운 값을 만들어야 합니다.
- 카드 합이 21을 초과하면 즉시 패배(버스트)합니다.
- 처음 두 장의 카드 합이 정확히 21이면 블랙잭입니다.

### 카드 점수 계산
- 숫자 카드(2~10): 카드에 표시된 숫자 그대로 계산
- J, Q, K: 각 10점으로 계산
- Ace: 1점 또는 11점으로 계산

### 플레이어 규칙
- 게임 시작 시 배팅 금액을 설정해야 합니다.
- 처음 두 장의 카드를 받은 후 다음 행동을 결정할 수 있습니다.
  - Hit: 카드를 한 장 더 받습니다.
  - Stand: 현재 카드로 승부를 겨룹니다.
- 21을 넘지 않는 한 원하는 만큼 카드를 받을 수 있습니다.

### 딜러 규칙
- 카드 합이 16 이하면 반드시 카드를 한 장 더 받아야 합니다.
- 카드 합이 17 이상이면 더 이상 카드를 받을 수 없습니다.

### 승패 및 배당
- 블랙잭: 처음 두 장의 합이 21인 경우, 배팅 금액의 1.5배를 획득합니다.
- 일반 승리: 딜러보다 21에 가까운 경우, 배팅 금액의 1배를 획득합니다.
- 무승부: 딜러와 금액이 같은 경우, 배팅 금액을 돌려받습니다.
- 패배: 딜러보다 점수가 낮거나 버스트한 경우, 배팅 금액을 잃습니다.
